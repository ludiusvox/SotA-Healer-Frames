SotA Healer Frames v1.0
======================
Creator: Ludiusvox
Version: 1.0

INSTALLATION:
1. Copy the 'SotaHealerFrames' folder to your Shroud of the Avatar Lua directory.
   (In-game type '/lua path' to find the folder).
2. Type '/lua reload' in-game to load the script.
3. Ensure the 'Lua Add-On Manager' has the script enabled.

NOTE: This script automatically hides the native party frames while active to prevent redundant UI. Disabling the script will restore them.

FEATURES:
- Centered Raid Frames: Tracks HP and Focus of party members.
- Specific Buff Tracking:
  * Soothing Rain: Bright Blue indicator (Top Left).
  * Dig In: Silver indicator (Top Right).
  * Torpor: Dark Green indicator (Bottom Left).
  * Healing Grace: Light Green indicator (Bottom Right).
  * Knight's Grace: Gold highlight in personal buff window.
- Debuff Tracking (Right side of frame):
  * Blind: Yellow indicator.
  * Fire DoT (Burning): Light Red indicator (use Douse).
  * Others (Stun, Root, etc.): Dark Red indicator.
- Status Cubes: Visual buff tracking with duration monitoring.
- Quest Ticker: Automatically updates when tasks are completed (captured via chat).
- Enhanced Combat Log: Categorized combat stream separate from social chat.

CONTROLS:
- Frames are informational only (click-to-heal is not supported by SotA API).
- Use F1-F8 keys to target players as indicated on the frames.
